Weather Task using Python ReAct Agent

## What It Does
This project is a minimal AI-assisted weather query agent built using Python.  
The agent accepts a user query related to weather, reasons step-by-step, calls a weather tool/function, and returns the final answer.

Example:
- User asks: "What is the weather in Kochi?"
- Agent processes the request
- Calls the weather tool
- Displays the weather response
